# Get-BskyTimeline

## SYNOPSIS

Get your Bluesky timeline.

## SYNTAX

```yaml
Get-BskyTimeline [[-Limit] <Int32>] [<CommonParameters>]
```

## DESCRIPTION

This command will return all items on your Bluesky timeline.

## PARAMETERS

### -Limit

Enter the number of timeline posts to retrieve between 1 and 100.
Default is 50.
